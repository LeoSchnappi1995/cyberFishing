# CyberFishing - 钓鱼游戏

单页面回合制策略钓鱼游戏，符合 GDD v3.1 规范。

## 使用方法

**直接打开 `index.html` 文件即可开始游戏！**

1. 双击 `index.html` 文件
2. 或在浏览器中打开该文件
3. 无需服务器，无需安装，开箱即用

## 游戏功能

### 核心玩法
- 🎣 **钓鱼系统**：选择水域、目标鱼类、鱼饵进行钓鱼
- 🏪 **商店系统**：购买船只、鱼饵、消耗品
- 🏆 **成就系统**：6个称号等待解锁
- 💾 **自动保存**：使用 localStorage 自动保存游戏进度

### 游戏规则
- **初始资金**：500元
- **船只升级**：无船 → 小船(300) → 中船(600) → 大船(1200)
- **成功率计算**：基础60% - (难度-1)×15% + 鱼饵加成 + 消耗品加成
- **翻船风险**：海湾(中船+难度5+成功率<40%)、深海(大船5%失败率)
- **深海阶段**：探索 → 完成(钓到大王乌贼) → 终极(钓到克苏鲁)

### 水域解锁
- **溪流/江边**：默认解锁
- **湖泊**：需要小船
- **海湾**：需要中船
- **深海**：需要大船

### 成就列表
1. 初心钓手 - 游戏开始
2. 海湾冒险者 - 首次进入海湾
3. 深海探路者 - 首次进入深海
4. 深海征服者 - 钓到大王乌贼
5. 深海猎人 - 钓到抹香鲸
6. 禁忌之触 - 钓到克苏鲁

## 游戏数据

游戏数据自动保存在浏览器的 localStorage 中，关闭浏览器后重新打开可以继续游戏。

## 技术说明

- 纯 HTML + CSS + JavaScript
- 无依赖，无需构建
- 单文件即可运行
- 使用 localStorage 进行数据持久化
